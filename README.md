# odin-library

This is a website for storing books that you're interested in.


The website is a book library where the user is able add, delete, edit, and keep track of books. Left-hand side of the site will have a sidebar for adding books and the entire site will be bookshelf where each book is added, starting from the top row, left to right. Users are able to click each book to get info on that book and the option to toggle if the book is read, edit and/or delete the book. When adding a book the user will be given a pop window, where they can input info of the book such as: name, author, # of pages, if read or not, etc.  

At the end of this project I will have structured and organized my Javascript code by using objects and prototypes. This also allows me to prevent code duplication and the ability to reuse code.
